# 栈

## 栈概念、结构、操作

一种后进先出（LIFO）的数据结构

栈底：栈的底部

栈顶：栈的顶部

入栈（push）：将数据存入栈顶

出栈（pop）：将数据从栈顶取出

栈顶元素（peek）：检测当前栈顶元素

特点：出栈和入都在栈顶操作

isEmpty：是否为空

clear：清空栈

size：栈元素的个数

## 栈操作实现

### 方法实现

| 方法名 | 操作 |
| :-: | :-: |
| push | 栈顶添加元素 |
| pop | 栈顶移除元素 |
| peek | 查看栈顶 |
| isEmpty | 检查栈是否为空 |
| clear | 移除全部元素 |
| size | 获取栈长度 |
