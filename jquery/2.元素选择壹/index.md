## .get(index)

获取指定DOM元素

- index为整数，按索引获取
- index为负数，倒叙获取
- 参数为空，获取DOM元素数组

```javascript
$('.demo').get(0)
$('.demo').get(-1)
$('.demo').get()
```

## .eq(number)

获取指定jQ元素

- index为整数，按索引获取
- index为负数，倒叙获取
- 参数为空，获取jQ空数组

```javascript
$('.demo').eq(0)
$('.demo').eq(-1)
$('.demo').eq()
```
