# 选择器

帮助你精准的选中想要的元素

## 简单选择器

1. ID选择器
2. 元素选择器
3. 类选择器
4. 通配符选择器

*：选中所有元素

5. 属性选择器

根据属性名和属性值选择元素

6. 伪类选择器

选择某些元素的某种状态

1）link：超链接未访问的状态

2）visited：超连接访问过后的状态

3）hover：鼠标移入状态

4）active：鼠标激活状态，鼠标按下

爱恨法则：love hate
