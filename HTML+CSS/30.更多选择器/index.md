## 更多伪类选择器

1. first-child

    选中第一个子元素
    
2. first-of-type

    选中元素中第一个指定类型的元素    

3. last-child

    选中最后一个子元素
  
4. last-of-type

    选中元素中最后一个指定类型的元素
    
5. nth-child

    选中指定位置的子元素，变量从 0 开始， 数字从 1 开始
    
    even：偶数 === 2n
    
    odd：奇数 === 2n + 1
    
6. nth-of-type

    选中指定的子元素中第几个某类型的元素
    
## 更多的伪元素选择器

1. first-letter

    选中元素中第一个文字
    
2. first-line

    选中元素中第一行的文字
    
3. selection

    选中被用户框选的文字
