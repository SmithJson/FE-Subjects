# a元素

超链接：从一个页面跳转到另一个页面

## href属性

hyper reference（引用）：表示跳转地址

1. 跳转地址
2. 跳转到某个锚点
3. 功能链接：点击后，触发某种功能
    + 执行js代码
    + 发送邮件mailto（要求用户计算机上安装有邮箱发送软件）
    + 拨号tel（要求用户计算机上安装有拨号软件或在移动端）

## id属性

全局属性，表示元素在文档中的唯一编号

## target属性

表示调转窗口位置

- _self：在当前页面窗口打开
- _blank：在新窗口代开

## title属性

全局属性，在鼠标悬浮在链接上时，显示提示文字
